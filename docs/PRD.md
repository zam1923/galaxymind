# PRD：GalaxyMind 統合プラットフォーム

> 最終更新：2026-02-16

---

## 1. 課題・背景

### 社会的課題

現代社会は「効率」と「正解」に支配されている。特に教育の現場では、探究学習が「形式的な調査と発表」に矮小化され、生徒が本来持つ**内発的動機**が起動する余地がない。

- **余白の喪失**：部活・受験・SNSに忙殺され、自分と向き合う「暇」がない
- **フィルターバブル**：アルゴリズムが「過去の自分の延長線」しか提示せず、偶然の出会いが阻害されている
- **視野の固定化**：狭い経験の中で将来を決めざるを得ず、世界の広さや選択肢の多さを知る機会がない

### プロジェクトの起点

イーロン・マスク氏の「銀河（Galaxy）レベル」の視座――既存の社会通念や短期的な効率性に捉われず、宇宙的なマクロの視点から人間の本質を捉え直すという思考法に感銘を受け、すべての事業と哲学を包摂する母体として **GalaxyMind** を立ち上げる。

## 3. ビジョン・ミッション

### GalaxyMind（最上位理念）

| 項目 | 内容 |
|---|---|
| **Vision** | 誰もが、自らの内なる好奇心に出会い、自分だけの夢を語れる社会 |
| **Mission** | 「模索」しているすべてのひとに、"無目的"な"余白"を提供する |

### Values（判断基準）

| Value | キーワード | 解説 |
|---|---|---|
| **暇（Schola）** | 精神的自由 | 余白こそが思考の苗床。自己と対峙するための「知のインフラ」を死守する |
| **偶然（Serendipity）** | フィルターバブル脱却 | 予期せぬ知性と衝突する「身体的な場」を設計する |
| **多層的な視座（Perspective）** | 世界の解像度 | 他者の知性や人生に触れ、人生の選択肢を正しく認識する |

### 模索に必要な3要素

| 要素 | 説明 |
|---|---|
| A. 時間 | 成果を問わない余白の時間 |
| B. 場所 | 偶然の出会いが起きる物理的・デジタル空間 |
| C. 方法 | 多様な情報と価値観に触れる媒介手段 |

### 模索手段の一覧

本（多様な知）/ 芸術（多様な感性）/ 対話（多様な考え方）/ 旅（多様な文化）/ 博物館（多様な歴史）/ 映画（多様な物語）/ クラブ活動（多様な役割）/ ボランティア（多様な社会の現実）/ 自然（多様な生命のサイクル）

## 4. 第一弾事業：BookMind

### BookMind 理念

| 項目 | 内容 |
|---|---|
| **Vision** | すべての生徒たちが、内なる好奇心に出会い、自分だけの夢を育み合える学びの場 |
| **Mission** | 学校図書館を「模索」の場へ。探究学習に「暇（余白）」と「偶然」を取り戻す |

### BookMind Values

| Value | 内容 |
|---|---|
| **暇（Schola）の保障** | 成果を問わない「読書と対話の空白時間」を教育カリキュラムに制度として組み込む |
| **偶然（Serendipity）の肯定** | 書架を歩く身体的経験を通じて、未知の興味を引き出す |
| **追体験（Reliving）** | 読書を「他者の人生の追体験」と定義し、世界の広さと選択肢を提示する |

### 好奇心のライフサイクル

| 教育段階 | 図書館の役割 |
|---|---|
| 小等教育 | 本を通じて世界を広げ、豊かな好奇心を「育む」土壌 |
| 中等・高等教育 | 多忙な日常に埋もれた本心を、本との対話を通じて「再発見」させる装置 |
| 共通 | 「自分だけの問い」に出会うことが目的 |

## 6. 競合・差別化

| 比較軸 | 既存サービス | GalaxyMind / BookMind |
|---|---|---|
| 目的 | 効率的な情報検索・学習管理 | 無目的な模索と偶然の出会い |
| UIの思想 | 機能重視・テキスト中心 | ビジュアル・ファースト、空間を感じるUI |
| 推薦方式 | アルゴリズムによるパーソナライズ | 身体的な偶然性（書架歩き）を重視 |
| 教育への姿勢 | 成果・評価ベース | 成果を問わない「余白」を制度化 |
| ポジショニング | EdTech / LibTech | **理念駆動型プラットフォーム**（思想が先、機能は後） |

### 直接競合が存在しない理由

本プロジェクトは「図書館管理システム」や「EdTech」ではなく、**思想・理念を起点とするブランド体験**を構築する。Apple Store が「商品を売る場所」ではなく「体験を提供する空間」であるように、GalaxyMindは「機能を提供するサービス」ではなく「模索という体験を提供するプラットフォーム」である。

## 7. UI/UX方針

### デザイン原則

| 原則 | 詳細 |
|---|---|
| **Apple Store的ミニマリズム** | 説明過多なテキストを排し、贅沢な余白と高精細なビジュアルで思想を伝える |
| **非言語的アプローチ** | 「文字を読む」前に「空間を感じる」UIを目指す |
| **カード型UI** | 情報を美しく陳列し、直感的にブラウズできる |
| **なめらかなインタラクション** | 階層構造の移動を体感できるトランジション |

### 情報階層

```
Identity層（GalaxyMind）
  └─ Category層（暇 / 偶然 / 視座）
       └─ Product層（BookMind / 将来の事業群）
```

### トーン & マナー

- 色調：深い宇宙的なダークトーンをベースに、アクセントカラーで光を表現
- タイポグラフィ：日本語は凛としたゴシック、英語はクリーンなサンセリフ
- 画像：抽象的・宇宙的なビジュアル、余白を贅沢に使う
- 言葉遣い：詩的で簡潔。説明ではなく「問いかけ」のトーン
